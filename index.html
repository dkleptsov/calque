How it works

  1. Drop a .docx / .xlsx / .pptx file into Bobr Translate.
  2. Pick a target language (or let auto-detect handle the source).
  3. Bobr Translate parses the document, extracts text run-by-run, sends each paragraph through the on-device translation model with format-preservation tags.
  4. Translated runs are written back into the original positions, preserving every byte of formatting.
  5. A translated copy appears next to the original: Report.ru.docx.
