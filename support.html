Frequently asked questions

Where does the translation actually happen?
Entirely on your Mac, using the Hy-MT1.5 translation model that you downloaded on first launch. Calque does not send your documents anywhere.
Why is the first launch so big (~1.1 GB)?
That's the offline translation model. We chose Tencent's Q4_K_M quantization for the right balance of size and quality. After the first download you never need internet again to translate.
Can I move the model file to an external drive?
Not directly — the macOS sandbox keeps the model inside the app's container. Workaround: go to Settings → Model variant and switch to a different quantization (Q4_K_M → Q6_K, Q8_0), or delete the existing file from ~/Library/Containers/com.calque.app/Data/Library/Application Support/com.calque.app/models/ and re-download.
Translation seems slow on my Intel Mac
Calque uses Metal acceleration where it's available. On Apple Silicon, expect ~20-50 paragraphs/sec; on older Intel Macs it falls back to CPU and is 5-10× slower. For Intel Macs we recommend the Q4_K_M model (default) — the larger quantizations are slower on CPU.
Some inline formatting (a bold word in the middle of a sentence) shifted
Calque uses the model's "formatted translation" prompt which keeps <sN> tags around each formatted run. In rare cases the model drops a tag — we fall back to proportional splitting based on character ratios, which can shift the formatting by a word or two. If you see this consistently, please attach the source file to a bug report so we can improve heuristics.
My language is not in the list
Calque supports the 36 languages of the Hy-MT1.5 model. Apple Russian/English/Chinese/Spanish/French/German/Japanese/Korean/Arabic/Portuguese/Turkish/Thai/Vietnamese and 23 more. We do not currently translate code-points outside this list. If a language is missing, drop us a note — we are watching upstream Tencent releases.
Can I use my own GGUF model?
Mac App Store policy forbids loading arbitrary executable code at runtime. Calque therefore ships a curated list of pre-verified model variants (visible in Settings → Model variant), each with a hardcoded URL and SHA-256 hash. We add new variants as they become available.
Does Calque work offline?
Yes, completely. The only network access ever is the one-time model download on first launch. After that, you can use the app in airplane mode indefinitely.
