Privacy Policy

Last updated: 13 May 2026

Bobr Translate is designed so the only place your documents go is your Mac. This policy explains, in plain English, exactly what the app does and does not do.

Short version

  • We do not collect, transmit, or store any of your data.
  • There is no account, no sign-in, no telemetry, no analytics, no advertising SDK.
  • Documents you translate never leave your Mac. Translation runs locally on Apple Silicon (Metal) or Intel CPU.
  • The only network request Bobr Translate ever makes is a one-time download of the translation model weights from huggingface.co on first launch.

What happens on first launch

To translate text on-device, Bobr Translate needs an offline translation model. The first time you open the app it downloads a single file:

  • a single model file (~1.1 GB), downloaded from huggingface.co.
  • The file is stored in your app's sandbox container at ~/Library/Containers/com.doclingua.app/Data/Library/Application Support/com.doclingua.app/models/.
  • A SHA-256 hash is verified after download so the file cannot be silently substituted.

After this one request, Bobr Translate never touches the network again. You can verify this by turning Wi-Fi off — translation continues to work.

What documents you translate

When you drop a .docx, .xlsx or .pptx file into Bobr Translate:

  • The file is read from the location you selected (the macOS sandbox grants access only to files you explicitly choose).
  • Text is extracted run-by-run and translated locally by the on-device model.
  • A translated copy is written back to disk next to the original (or to a folder you choose) with a language suffix, e.g. Report.ru.docx.
  • No copy of your document is sent over the network, stored remotely, or logged. Nothing is retained by Bobr Translate outside the file you can see in Finder.

Sandbox & entitlements

Bobr Translate runs under the macOS App Sandbox. The full list of permissions it requests:

  • com.apple.security.app-sandbox — sandboxing on.
  • com.apple.security.network.client — to download the model weights (one-time).
  • com.apple.security.files.user-selected.read-write — to read the file you drop in and write the translated copy.
  • com.apple.security.files.bookmarks.app-scope — to remember recently-translated files between launches.

That is the complete list. Bobr Translate cannot reach any other part of your file system, your contacts, calendar, or device sensors.

Children

Bobr Translate does not knowingly collect data from anyone, including children under 13.

Changes to this policy

If the policy ever changes materially, the new version will be posted here and the "Last updated" date will be revised.
